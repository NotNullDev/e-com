# Sample online shop

> ***Work in progress***.

Made with Next.js, TailwindCSS, Prisma, DaisyUI, TRPC and Stripe.

Bootstrapped with create-t3-app.


## Dev commands

```bash
# run dev containers
docker compose -f ./docker-compose-dev.yml up -d --force-recreate
```
